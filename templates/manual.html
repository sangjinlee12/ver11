시스템 개요

에스에스전력 인사관리시스템은 직원들의 휴가 신청 및 승인, 재직증명서 발급을 효율적으로 관리하는 웹 기반 시스템입니다.

휴가 관리

온라인으로 휴가를 신청하고, 관리자의 승인 처리 현황을 실시간으로 확인할 수 있습니다.

증명서 발급

재직증명서를 온라인으로 신청하고 PDF 형태로 즉시 다운로드할 수 있습니다.

현황 조회

개인별 휴가 사용 현황과 잔여 휴가를 쉽게 확인할 수 있습니다.

모바일 지원

PC와 모바일에서 모두 사용할 수 있는 반응형 웹 디자인을 제공합니다.

로그인 방법

시스템 접속

웹 브라우저에서 시스템 주소로 접속합니다.

로그인 정보 입력

사용자명과 비밀번호를 입력합니다. 초기 비밀번호는 관리자에게 문의하세요.

로그인 버튼 클릭

"로그인" 버튼을 클릭하여 시스템에 접속합니다.

로그인 문제 해결

아이디나 비밀번호를 잊어버렸을 경우, 로그인 페이지의 "아이디/비밀번호 찾기" 링크를 이용하거나 관리자에게 문의하세요.

휴가 신청 및 관리

휴가 신청 방법

휴가 신청 메뉴 선택

메인 대시보드에서 "휴가 신청" 버튼을 클릭합니다.

휴가 정보 입력

휴가 시작일, 종료일, 휴가 종류(연차, 반차, 병가 등), 사유를 입력합니다.

신청서 제출

입력한 정보를 확인하고 "신청" 버튼을 클릭합니다.

휴가 현황 조회

신청 내역

본인이 신청한 휴가의 승인 상태를 확인할 수 있습니다.

휴가 잔여일

연차 총 일수와 사용한 휴가, 남은 휴가를 확인할 수 있습니다.

휴가 신청 시 주의사항
  • 휴가는 최소 3일 전에 미리 신청해 주세요.
  • 공휴일과 주말은 자동으로 제외되어 계산됩니다.
  • 반차는 오전/오후 중 선택할 수 있습니다.
  • 승인 후 휴가 취소는 관리자에게 문의하세요.

재직증명서 발급

증명서 신청

메인 대시보드에서 "재직증명서 신청" 버튼을 클릭합니다.

사용 목적 입력

재직증명서가 필요한 용도(은행 대출, 관공서 제출 등)를 입력합니다.

신청서 제출

"신청" 버튼을 클릭하여 재직증명서 발급을 신청합니다.

증명서 다운로드

승인 후 "다운로드" 버튼을 클릭하여 PDF 파일로 다운로드합니다.

전자문서

바코드가 포함된 전자문서 형태로 발급되어 진위 확인이 가능합니다.

즉시 발급

관리자 승인 후 즉시 PDF 형태로 다운로드할 수 있습니다.

관리자 기능

관리자 권한이 있는 사용자만 접근할 수 있는 기능들입니다.

직원 관리

신규 직원 등록, 정보 수정, 엑셀 일괄 업로드 기능을 제공합니다.

휴가 승인

직원들의 휴가 신청을 검토하고 승인/반려 처리할 수 있습니다.

증명서 관리

재직증명서 신청을 승인하고 직접 발급할 수 있습니다.

통계 조회

전사 휴가 현황, 부서별 통계를 확인할 수 있습니다.

관리자 메뉴 접근 방법

관리자 로그인

관리자 권한이 있는 계정으로 로그인합니다.

관리자 메뉴 선택

상단 네비게이션에서 "관리자" 메뉴를 클릭합니다.

세부 기능 선택

직원 관리, 휴가 관리, 증명서 관리 중 필요한 메뉴를 선택합니다.
